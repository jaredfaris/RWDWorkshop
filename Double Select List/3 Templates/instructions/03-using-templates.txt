HTML in our JS is icky. Let's use a template instead.

Add a reference to handlebars.js to the <head>

    <script src="js/handlebars-v1.3.0.js"></script>


Create a script block for the template.
QUESTION: Where does this need to be on the page?

    <script id="dslOptionsTemplate" type="text/x-handlebars-template">
        // template markup here
    </script>


Look at the examples to figure out what to put in this template.


If you don't already have one, create one or more arrays (depending on your solution) of JS objects that we can map to them.
Something like:

 [{name: "SomeName", value: "SomeValue"}, â€¦]


Read the template in and compile it:

   var templateHTML = $('#myTemplate').html();
   var template = Handlebars.compile(templateHTML);


Use the template with your data:

   var myNewHTML = template(myData);


Write the new HTML to the page:

    $('#someTarget').append(myNewHTML);
