Add a reference to Knockout.js in the <head>

    <script src="js/knockout-3.1.0.js"></script>

Remove our handlebars template and instead add the KO template inline with its container
Use the data-bind foreach attribute to have it loop through options:

    <select multiple class="unselected-values" data-bind="foreach: items">
        // more KO markup will go here
    </select>


Take our initial array of objects and make the selected property an observable:

    var myItems = [
        {value: 1, name: "Value 1", selected: ko.observable(false)},
    ]

Create a viewModel with this array as an observableArray:

    var myViewModel = {};
    myViewModel.items = ko.observableArray(myItems);


Apply the KO binding to somewhere in your HTML using this view model:

    ko.applyBindings(myViewModel, document.getElementById("myDiv"));

    or

    ko.applyBindings(myViewModel, $('#myDiv')[0]);

Update your arrow buttons to find an item in the items list and update its selected property:

    foundItem.selected(true)
    or
    foundItem.selected(false)

Watch KO magically update your page.
